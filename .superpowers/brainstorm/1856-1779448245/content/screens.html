App Screens

What each user sees when they log in

👷 Worker — 4 screens

1. Dashboard (Home)
Current balance (big ₹ number) · Low balance warning banner · Recent expenses list · Quick "Add Expense" button
2. Add Expense
Amount · Category (dropdown, mandatory — "Other" always listed) · Comment (text field) · Upload photo (optional) · Submit
3. Expense History
Full list of own expenses · Filter by category or date · Tap to see receipt photo and comment
4. My Categories
See global categories (view only) · Add / delete own custom categories

👑 Owner — 5 screens

1. Dashboard (Home)
All workers listed · Each shows current balance · Low balance workers highlighted in red · In-app low balance alerts
2. Worker Detail
Worker's balance · Fund transfer history · Full expense history · "Add Funds" button · Reset email/password
3. Workers Management
Add new worker (name + email + temp password) · Deactivate worker
4. Reports
Filter by worker / category / date range · Summary totals · Export as CSV or PDF
5. Settings
Low balance threshold (₹) · Manage global categories (add / delete, "Other" undeletable) · Owner email for alerts
