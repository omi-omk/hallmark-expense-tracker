Key Flows

How the important actions work end-to-end

🔐 Auth Flow

Owner creates worker account (name + email + temp password)
→ Worker receives email with temp password
→ Worker logs in, encouraged to change password
→ If worker forgets credentials → owner resets from Worker Detail screen
→ Owner account is seeded once at setup (not self-registerable)

💸 Fund & Expense Flow

Owner sends real money to worker (outside app)
→ Owner opens Worker Detail → clicks "Add Funds" → enters ₹ amount + note
→ Worker's balance updates immediately
→ Worker logs in → sees updated balance
→ Worker adds expense → balance deducted automatically
→ If balance drops below threshold → owner gets in-app alert + email

📊 Reports & Export Flow

Owner opens Reports → filters by worker / category / date range
→ Sees table: worker name · date · category · amount · comment
→ Clicks Export CSV → downloads spreadsheet
→ Clicks Export PDF → downloads formatted report

🔔 Low Balance Alert Flow

Every time a worker records an expense:
→ App recalculates their balance
→ If balance < threshold → send email to owner via Resend
→ Show red badge on that worker's card on owner dashboard
→ Alert dismissed when owner adds funds
